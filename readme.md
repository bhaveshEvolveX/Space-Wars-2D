# Space Wars (V1.0) 🌌🎮

Welcome to **Space Wars**, a classic, fast-paced 2D arcade space shooter built completely from scratch using the **Godot Engine 4.6.2** and **GDScript**. 

Take control of your spaceship, dodge incoming enemy attacks, and blast your way through space to achieve the highest score possible!

---

## 🚀 Features

*   **Classic Arcade Mechanics:** Fluid player movement, responsive weapon shooting, and dynamic scoring.
*   **Dynamic Spawning:** Face off against waves of randomized enemy ships that test your reflexes.
*   **Fully Functional Menus:** Features a seamless game flow from the Main Menu, to a fully working Settings/Audio configuration screen, down to the Game Over screen.
*   **High Score Tracking:** Keeps track of your personal best so you always have a target to beat.

---

## 🛠️ Built With

*   **Engine:** Godot Engine (v4.6.2)
*   **Language:** GDScript
*   **Assets:** 2D Retro Pixel Art & Classic Sound Effects

---

## 🎮 How to Play

1. **Clone the Repository:**
```bash
   git clone [https://github.com/bhaveshevolvex/Space-Wars-2D.git](https://github.com/bhaveshevolvex/Space-Wars-2D.git)
2. Open in Godot: Import the project folder directly into your Godot Engine project manager.

3. Run the Project: Hit F5 to play!

Controls:

Use your mouse to navigate menus.

[Move Left - A / LeftArrow
Move Right - D/ RightArror]

[Shoot - Spacebar / LeftMouseButton]
